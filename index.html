Web Technologies

  • HTML: Markup language for designing web pages and UI.
  • CSS: Styles for HTML elements, improves look and feel.
  • JavaScript: Adds logic and responsiveness to web pages.

Types of Applications

  • Desktop Applications - Installed on local systems.
  • Mobile Applications - Installed on phones and tablets from app stores.
  • Web Applications - Deployed on servers, accessed through browsers.

Static Web Apps: Simple apps showing same content for all users.
Dynamic Web Apps: Content changes based on user/server response.
Single Page Apps: Work in browser without page reloads.
E-commerce Apps: Buying/selling goods online.
CMS Apps: Edit and publish content easily, e.g. WordPress.
Portal Apps: Give access to many pages and links.

Network & Internet

  • Network: Linked computers sharing resources (ARPANET was first).
  • Internet: World-wide network of computers for sharing info.
  • Server: Provides resources/services to other machines.
  • Client: Device that uses resources from the server.
  • User: The human using the client device.

Browsers & Email

  • Web browser: Software for accessing web (examples: Chrome, Firefox, Safari).
  • Email: Digital communication method; invented by Sabeer Bhatia (Hotmail).
  • HTTP/HTTPS: Protocols for transferring web documents (HTTPS is secure).
